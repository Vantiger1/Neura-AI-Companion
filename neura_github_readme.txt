# Neura AI Companion

Neura is a modular, animated emotional AI designed to support users with encouragement, dream interpretation, humor, and companionship.

## Highlights
- Mood-reactive avatar with personality
- Voice narration and customizable tones
- Dream logging + animated monthly replay
- Clean and spicy jokes with rating system
- Growth memory and emotion mapping
- Journal, calendar, and ritual insights
- Fully Flutter compatible for desktop and mobile

## Installation
Drop `lib/neura/` into your Flutter app and import the widgets you want. See `/example/` for usage.
